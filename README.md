# cvwo-dota-forum
Project for CVWO

Setup:

Prerequisites:

Node.js and npm

Yarn package manager

Ruby on Rails v7

Postgresql


Setting up the database

Run the following commands:

yarn

rake db:create

rake db:migrate

bin/dev

View the website at
localhost:3000
